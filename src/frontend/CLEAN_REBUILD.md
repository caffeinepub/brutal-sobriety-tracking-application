# BRUTAL App - Clean Rebuild Guide

This guide explains how to perform a clean rebuild of the BRUTAL frontend to resolve deployment issues caused by stale build artifacts or corrupted caches.

## When to Use Clean Rebuild

Perform a clean rebuild when you encounter:
- Deployment failures without clear error messages
- Build timeouts or hanging builds
- Stale TypeScript errors that don't match your code
- Dependency resolution conflicts
- Canister install failures on the Internet Computer
